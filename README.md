# Pong Game

A classic Pong game implementation running directly in the browser.

## How to Play

1. Open `pong2.html` in your web browser.
2. Control the paddles to hit the ball back and forth.
3. Prevent the ball from passing your paddle!

## Usage

Simply clone the repository and open the HTML file:

```bash
git clone https://github.com/littlebitsrodos/pong-game.git
cd pong-game
open pong2.html
```
